## Financial Econometrics
Financial Econometrics combines economic theory, finance, statistics, and machine learning to model financial markets and asset prices. 
This project provides practical implementations of key financial econometric techniques for:
- Time-series forecasting
- Asset return modeling
- Volatility forecasting
- Risk management

#### Time Series Analysis
- AR, MA, ARMA, ARIMA models
- SARIMA model
- VAR model
  
#### Volatility Modeling
- ARCH model
- GARCH model
- Stochastic volatility models

#### Asset Pricing
- CAPM
- Fama-French Factor Model
- Multi-factor asset pricing model

#### Risk Management
- Value at Risk (VaR)
- Expected Shortfall (ES)
- Monte Carlo simulation
- Stress testing
- Backtesting procedures


#### Data sources:
- Yahoo Finance
- FRED Economic Database
